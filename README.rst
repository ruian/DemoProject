How to Install

``` ./bin/vendors update ```

Doctrine Fixture
================

Dependances :

 * [doctrine-fixtures](http://github.com/doctrine/data-fixtures)
 * [DoctrineFixturesBundle](http://github.com/doctrine/DoctrineFixturesBundle)
